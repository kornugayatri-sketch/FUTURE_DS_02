# FUTURE_DS_02
# Customer Retention & Churn Analysis

# Project Overview

This project analyzes customer churn behavior using a telecom customer dataset.
The goal is to understand why customers leave a service and identify factors that help improve customer retention.

This task is part of the Data Science & Analytics Internship Task-2 by Future Interns.

# Objectives

Analyze customer churn patterns

Identify factors influencing churn

Study customer tenure and retention trends

Provide insights to improve customer retention

# Dataset

Dataset used: Telco Customer Churn Dataset

Main features in the dataset include:

Customer demographics

Contract type

Payment method

Monthly charges

Total charges

Customer tenure

Churn status

Total records: 7043 customers

Data Analysis Steps

Data Loading

# Key Insights

Customers with month-to-month contracts have the highest churn rate.

Customers with short tenure are more likely to leave.

Higher monthly charges may influence churn behavior.

Long-term contracts help improve customer retention.

# Visualizations

The project includes visualizations such as:

Churn distribution

Churn by contract type

Tenure vs churn density plots

Business Recommendations

Encourage customers to choose long-term contracts.

Provide discounts or loyalty benefits for long-tenure customers.

Improve services for customers with high monthly charges.
